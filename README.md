>    3. 3. Paste into any AI (ChatGPT, Claude, Gemini)
>       4. 4. Get professional results in 60 seconds
>         
>          5. ## The RACE Framework
>         
>          6. Every prompt follows **RACE**:
>          7. - **R**ole - Tell the AI who to be
>             - - **A**ction - Tell it what to do
> - **C**ontext - Give all relevant info
> - - **E**xpected output - Define the format
